## Come usare

1. Apri `index.html` nel browser
2. Modifica i dati in `data/data.json` per aggiornare:
   - Risultati partite
   - Classifica
3. I dati si salvano automaticamente in localStorage
4. Le sezioni si aggiornano in tempo reale
